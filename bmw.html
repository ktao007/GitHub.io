BMW Finance CFE online application

Digitization is transforming the way that business is conducted in China. ReignDesign help BMW China Finance to transform its channels, processes and service into an online application in the digital age.

This project is conducted by team work. I created the UX for the project together with my mentor Jason Wong and Joshua Bradley. I designed the interface for the web mentored by Andreas Sundgren and Shinae Kang.





All the applications information and documents upload and reviewed process should be deisgned into this digital UX process.





My mentors created the IA based on the features of this project. Then I writed down the UX documents for some of the screens. The documents include solid infomation for each screen which asked us to think clearly in the abstract level.


The user flow for Register and Login part. Its more complicated for Forget password for this App because of high security requirement.

for stakeholders

There are Main Applicant, Co-Applicant and Guarantor involved in this application. The overal goal is to help them easily take part in the application process, finish they tasks and follow up for the updates. It turn out they will be 15 differents scenerio for the home page notification states.


The user flow for application process.
The four steps include:
1-Selecting vehicle and SF products;
2-Privacy data authorization and PBOC record checking;
3-Personal infomation filled;
4-Documents upload and upselling.


Based on the IA, user flow and UX documents, we start to create the wireframe.


I work on 1/3 of the mobile wireframes.
And 1/2 of the Web wireframes.


As BMW have a strict Branding Guildeline, we need to apply this Website and App guildeline into the final UI screens.

During the adaption, I found there are many cases in this project that the guildeline doesn't cover, then I took a wide research for BMW website among other countries and found the userful design pattens.


I designed all the Web screens under the instrutions of my mentors.
